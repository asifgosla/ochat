# OChat

## Overview

An application to chat with an LLM using Ollama.

## Technologies

- C++, C++17
- Bazel
- REST API client (using cpp-httplib)
- JSON (using boost::json)
- Ollama

## Dependencies

- cpp-httplib
- boost::json
